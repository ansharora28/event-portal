SRM Events Portal

Event organizers' one-stop solution for event reminders!




How it works?

  1. Create an event.📆
  2. Check upcoming events page for event created succesfully.✅
  3. Event invites are sent to organiser's email in 4-5 minutes.📩
  4. Respond yes to calendar invite.✔️ You can modify event details in google calendar, choose when you want to recieve the notification!📝
  5. Reminder notification sent on chosen date on all devices where google calendar is logged in!📨
